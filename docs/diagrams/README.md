# QuantIAN Architecture Diagrams

This directory contains the editable `draw.io` source files for the QuantIAN architecture deliverables.

Current diagram set:

- `quantian_local_mvp.drawio`: detailed local MVP runtime view
- `quantian_final_cloud.drawio`: detailed hybrid cloud deployment view for the live AWS VM, Azure Container Apps, and GCP VM rollout

Diagram metadata standard:

- Course: Cloud Computing
- Professor: Jean-Claude Franchitti
- Institution: NYU Courant - New York University
- Authors: Bhavesh, Aditi, Anusha
- State: To-Be
- Status: Working Draft

Recommended export workflow:

1. Open the `.drawio` file in diagrams.net / draw.io.
2. Review labels, cloud regions, and service names against the current VM deployment notes.
3. Export PNG copies for slides after final review.
